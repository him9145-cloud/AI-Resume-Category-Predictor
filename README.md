# AI Resume Category Predictor

An NLP and Machine Learning project that classifies resumes into job categories based on resume content.

## Features
- Resume classification using TF-IDF and Machine Learning
- 97.93% prediction accuracy
- Real-time predictions with Streamlit
- Text preprocessing and feature extraction

## Tech Stack
Python, Pandas, Scikit-learn, NLP, Streamlit

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
